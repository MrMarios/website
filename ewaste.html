Η άποψη μου για το E-waste

Εδώ πιστεύω ότι κάποιος μπορεί να μπερδευτεί για το τι πραγματικά θέλω να μιλήσω,οποτε θα ξεκαθαρισω.Οταν μιλάω για E-waste εννοω οποιαδήποτε τεχνολογια που λειτουργεί με μπαταρία ή ρεύμα που πετιέται στα σκουπίδια.

Ανοίγω αυτό το θεμα καθώς πιστεύω πως ειναι τρομερό το γεγονός ότι καποιοι άνθρωποι πετάνε τεχνολογία που λειτουργεί κανονικά αλλά οχι τέλεια.Οταν εφευρέθηκε το κομπιουτερακι για τις απλές αριθμητικές πράξεις οι άνθρωποι είχαν πάθει πλάκα, είχαν τρελαθεί πως θα μας κάνει χαζούς και τέτοια,και τωρα πια όταν κάποιος πετάει τον υπολογιστή του επειδή ειναι αργός ή μπαταρία εχει χαλάσει κανεις δεν λέει τίποτα.Περα απο το γεγονός ότι όλη αυτή η τεχνολογία εχει πολυ κακο αντίκτυπο στο περιβάλλον εγω θέλω να πω για το γεγονός πως πολυ κόσμος ειναι υπολογιστικά αναλφάβητος.Δεν κατανοήει τι πραγματικά συμβαίνει στον υπολογιστή.

Θα πω για τη μπαταρία στα laptop,κινητά κλπ όλες τις μπαταρίες που ειναι ΛΙΘΙΟΥ,οι μπαταρίες λίθιου ειναι ευαίσθητες απο μόνες τους και οταν τις κακομεταχειριζομαστε υπαρχουν άσχημα αποτελέσματα,ο χειρότερος εχθρός αυτών των μπαταριών ειναι η ζέστη,δεν πρέπει με τιποτα να χτυπάει ο ήλιος οποιαδηποτε συσκευη που εχει μεσα τετοια μπαταρία.

Επιπλέον ενα αλλο συχνό λάθος ειναι που πολυ κόσμος δεν βγάζει το καλώδιο όταν η συσκευή ειναι στο 100%, κατι καταστρέφει πολυ γρήγορα την συσκευή.Δυστηχως οποία βλάβη εχει γινει στη μπαταρία δεν αντιστρέφεται(εκτός αν αλλαχθεί που δεν αξιζει).

Τωρα θα πω για τους υπολογιστές πιο συγκεκριμένα,ακούω συχνά τη φράση ο υπολογιστής μου ειναι αργός.Αυτο συμβαίνει οχι εξαιτιας του υπολογιστή αλλα του χρήστη.

Αν ο υπολογιστης ειναι πολυ παλιός και εχει HDD αντι για SSD τοτε αυτο μπορει να παίζει μεγάλο ρόλο.Επισης ειναι σημαντικό να αναγνωρίσουμε το γεγονός πως υπάρχουν ιοί και αν δεν προσέξεις μπορει να έχεις κάποιον στο συστημα χωρίς να το καταλάβεις προτείνω την εγκατάσταση του Malwarebytes ώστε να καταλάβεις αν έχεις πρόβλημα με ιούς.Κατι επίσης σημαντικό ειναι πως συχνα οι υπολογιστες δεν έχουν χώρο.Αν το C: ειναι στο κόκκινο τότε υπάρχει πρόβλημα πρέπει να διαγραφούν αρχεία είτε προγραμματα που δεσμεύουν πολυ χώρο.Επιπλεον ο υπολογιστης δεν εχει απεριόριστη μνήμη πολλές φορές αφήνουμε προγραμματα ανοιχτά που δεσμεύουν πολυ μνήμη και δεν τα χρησιμοποιούμε καν.Ενα ισχυρό παράδειγμα ειναι το chrome και κάθε του παραλλαγή.Το chrome χρησιμοποιεί παρα πολυ μνήμη παρόλο που ειναι ενα browser Αν πατήσεις το ctrl shift esc στο πληκτρολόγιο θα σου βγάλει το task manager αν πας στο processes πατήσεις το Name και κοιτάξεις αριστερά μπορεις να δεις ακριβώς ποια προγράμματα καταναλώνουν πολυ μνήμη κατω απο το Memory.Θελω επισης να τονίσω και κάτι άλλο που μπορεί να αλλάξει την οπτική σε πολυ κόσμο για τα σύγχρονα windows.Τα windows 10 και 11 έχουν πολλές προεγκατεστημενες εφαρμογές που απαιτούν μνήμη και δυστυχώς έτσι μειώνεται η αποδοτικοτητα ενός συστήματος.Πιστευω πως αν τα windows ειναι πραγματικά πολυ βαριά για ενα υπολογιστη τότε τα linux θα σώσουν ενα τέτοιο συστημα.Τα linux σε αντίθεση με τα windows τρέχουν από πισω μόνο τα απολύτως απαραίτητα που χρειάζονται. Για μια ομαλή χρηση του συστηματος τα windows θέλουν 8 GB μνήμης ενώ τα linux θέλουν μόνο 4GB μνήμης. Παρολαυτα όμως για να μπορείς να χρησιμοποιήσεις linux χρειάζεσαι μία x γνώση στους υπολογιστές.Παρολαυτα linux distros οπως LinuxMintXFCE και Lubuntu ειναι πολυ καλές επιλογές για κάποιον που θέλει να υπάρχει ένας υπολογιστης
